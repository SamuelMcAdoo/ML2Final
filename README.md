## Purpose

This project aims to detect and correctly solve CAPTCHA images. 

## Dataset(s)
The initial dataset used is from Kaggle, and contains 1,070 anotated CAPTCHA images: https://www.kaggle.com/datasets/mrigaankjaswal/capcha-images-to-training-data . This dataset is included with this repo, and would work fine for a limited and low quality training of this project. 

The dataset used in reality to obtain good accuracy is found at https://www.kaggle.com/datasets/parsasam/captcha-dataset , also on Kaggle, and containing 113,000 annoted CAPTCHA images. This dataset is not included in this repo as it is too large. However, if you wish to run the notebook, you can download it from the aforementioned link and upload it manually to Colab. This can take some time depending on the speed of your internet. It takes me about 5 minutes, but may take you much longer.

## Running this Notebook
In order to run this notebook, it is suggested that you use an A100 on Google Colab, as I have. simply open the notebook included in this Git, upload a working dataset (I recommend the dataset which utlizes 113,000 CAPTCHA images if you want good results.)
